# Exodar

Meteor Client addon.

## Versions

| Minecraft Version | Exodar version | Dependencies |
|-------------------|----------------|--------------|
| 26.1.2 | [1.0.1](https://github.com/ShaunaAyala/Exodar-Addon/releases/download/v1.0.1/Exodar-Addon-1.0.1-26.1.2.jar) | Fabric Loader 0.19.2+, Meteor Client 26.1.2 |
| 1.21.11 | [1.0.1](https://github.com/ShaunaAyala/Exodar-Addon/releases/download/v1.0.1/Exodar-Addon-1.0.1-1.21.11.jar) | Fabric Loader 0.16.10+, Meteor Client 1.21.11 |

## Features

**Modules** (Exodar category):
- ESprint
- ETool
- AimAssist
- TriggerBot
- AutoClicker
- LegitAura
- Penetration

## Install

Drop the JAR into your Fabric mods folder alongside Meteor Client.

## Source

- `main` branch — Minecraft 26.1.2 source
- `1.21.11` branch — Minecraft 1.21.11 source

## Build

```
./gradlew build
```

Output: `build/libs/exodar-addon-1.0.1.jar`
